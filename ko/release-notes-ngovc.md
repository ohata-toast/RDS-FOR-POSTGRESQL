## Database > RDS for PostgreSQL > 릴리스 노트

### 2026. 02. 10.

#### 기능 추가

- DB 확장 기능 추가
    - `pg_cron` 확장을 추가했습니다.
- 데이터베이스, 스키마 권한 관리 기능 추가
    - 권한이 있는 사용자만 데이터베이스, 스키마에 접근할 수 있도록 관리하는 기능을 추가했습니다.
    - DDL 권한이 있는 사용자에 한해 데이터베이스, 스키마의 소유자로 지정할 수 있습니다.
- Resource Watcher 서비스 연동
    - Resource Watcher 서비스에서 DB 인스턴스 리소스를 확인할 수 있도록 기능을 추가했습니다.

#### 기능 개선

- 시점 복원 정보 조회 기능 개선
    - WAL 로그 파일이 많이 쌓여 있을 때의 조회 속도를 개선했습니다.
- 사용자 권한 확대
    - 읽기 전용 권한인 READ 권한을 추가했습니다.
    - DDL 권한에 대한 제약을 완화하여 추가 생성하거나, 기존 사용자 권한을 DDL로 변경하거나 삭제할 수 있습니다.

#### 버그 수정

- 고가용성 인스턴스 접근 제어 규칙 불일치 문제 수정
    - 특정 상황에서 이전에 적용한 접근 제어 규칙이 마스터와 예비 마스터 간 불일치하던 문제를 수정했습니다.
- 삭제 불가 사용자가 삭제된 것으로 표시되는 문제 수정
    - 객체를 소유하고 있어 삭제가 불가능한 사용자가 콘솔에서는 삭제된 것으로 잘못 표시되는 문제를 수정했습니다.
    - 객체를 소유한 사용자 삭제 시 추가 옵션을 선택하여 정상적으로 삭제할 수 있도록 개선했습니다.

### 2025. 10. 28.

#### 신규 서비스 출시

- Relational Database Service(RDS)는 관계형 데이터베이스를 클라우드 환경에서 제공하는 서비스입니다.
- 복잡한 설정 없이 관계형 데이터베이스를 사용할 수 있습니다.
